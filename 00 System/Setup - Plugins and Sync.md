# Plugins and Sync

## Enabled Core Plugins

- Daily notes
- Templates
- Unique note creator
- Search
- Properties

## Recommended Community Plugins

Install these from the Obsidian community plugin browser:

1. Calendar
   Useful sidebar calendar for fast navigation between daily notes.
2. Tasks
   Good if project action items start spreading across daily, meeting, and project notes.
3. Git
   Optional for commit/push from inside Obsidian, but not a replacement for live sync.

## Recommended Mac Sync

For Linux to Mac automatic syncing, prefer one live sync method:

- Obsidian Sync
- Syncthing

Use Git in parallel for history, backup, and publishing to GitHub.

Do not depend on Git alone as your "automatic sync" layer for daily note-taking.

## AI-Friendly Rules

- Keep one concept per note when possible.
- Use YAML properties for structured metadata.
- Prefer descriptive filenames.
- Link related notes explicitly with `[[wiki links]]`.
- Keep raw source material and permanent notes separate.
