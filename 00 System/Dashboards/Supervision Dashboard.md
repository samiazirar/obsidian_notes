# Supervision Dashboard

## Thesis Notes

Create one note per student or thesis in [[03 Research/Supervision]].

Track:

- topic and scope
- meeting cadence
- current blockers
- next action before the next meeting

## Suggested Pattern

1. Keep one main supervision note per thesis.
2. Put detailed meeting notes in [[05 Meetings]].
3. Link every meeting note back to the thesis note.
