# obsidian_notes

An Obsidian vault for project-based PhD work.

## Structure

- `00 System/` vault dashboard, templates, attachments
- `01 Daily/` fast daily notes
- `02 Projects/` active research and technical projects
- `03 Research/Literature/` papers, reading notes, bibliographies
- `03 Research/Supervision/` master thesis supervision notes
- `04 Ideas/` rough concepts and future work
- `05 Meetings/` meeting notes across projects
- `99 Archive/` closed or inactive material

## Sync Strategy

Recommended:

1. Use Git for version history and publishing.
2. Use a live sync tool between machines for automatic file sync.

Best cross-platform choice for Linux <-> Mac:

- `Syncthing` or `Obsidian Sync`

Avoid running two different live sync tools on the same vault at once.
Git is fine alongside one live sync method.
